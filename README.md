## Banking Application Test Automation Framework

A professional QA automation framework built with Python, Playwright and pytest.

### Tech Stack
- Python 3.11
- Playwright 1.58
- pytest 9.0
- GitHub Actions CI/CD

### Test Coverage
- ✅ UI Testing — Guru99 Bank (Login, New Customer)
- ✅ API Testing — REST API (GET, POST, PUT, DELETE)
- ✅ Data Driven Testing — Excel driven test data
- ✅ CI/CD — Automated pipeline with GitHub Actions

### Running Tests
```bash
pytest -v
```
